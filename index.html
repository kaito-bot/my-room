My Creations

02

Web Development

I prefer to learn by doing. So, throughout my academic years, I have developed many web-based projects that helped me enhance my skills and provide me better understanding of different technologies. Do check it out 😉!

Utopia

"Utopia Fashion E-Store" is a vibrant online platform catering to the diverse fashion needs of families. To develop this application, I used React.js, Express.js, Redux, Firebase, SCSS and Stripe API's payment gateway.

VS 2.0

"VS 2.0" is a 3D Room Portfolio Website. This was the first time I have worked with 3D and I absolutely loved it ❤️. For this website, I developed a 3D model using Spline and used Three.js (to render the model and environment),GSAP (to animate things) and Vanilla javaScript to develop this beautiful Room Portfolio Website.

Evento

This project is a work in progress 🔧, but it is an event management application that will be developed using MERN stack to streamline the process of event creation and registration.

Monsterque

"Monsterque" is a simple search component featuring an elegant and intuitive User Interface. Through seamless integration with the Fetch API, Monsterque swiftly retrieves data from a third-party API, enabling users to swiftly locate their desired character. I used React.js to develop this small application.

HaveSomeWater

"HaveSomeWater" is a small web application that keeps track of how much water you drank in the whole day and how much water💧 is remaining. I developed this small application using HTML, CSS and JavaScript

Finansiera

"Finansiera" is a 🪙 Finance company's landing page. I Developed this website using HTML and CSS.

Kavai

"Kavai" is a Hotel's landing page. I Developed this website using HTML and CSS.
